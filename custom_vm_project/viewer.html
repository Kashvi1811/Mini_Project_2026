How To Run Your Program
1) Write or edit code in C++/ASM editor.
2) Press Compile & Flash once after every code change to load new instructions.
3) Press Next Step repeatedly to execute step-by-step (Fetch → Execute).
4) Use Auto Run for automatic stepping, and Back to undo one step.
Important: If you edit code again, press Compile & Flash before stepping.
