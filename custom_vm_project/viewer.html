How To Run Your Program
1) Write or edit code in High-Level/ASM editor and choose your language from the dropdown.
2) Press Compile & Flash once after every code change to load new instructions.
3) Press Next Step repeatedly to execute step-by-step (Fetch → Execute).
4) Use Auto Run for automatic stepping, and Back to undo one step.
Important: If you edit code again, press Compile & Flash before stepping.

IR Field Guide
Opcode (4 bits): which operation to run (ADD, LOAD, JMP...).
Dest (3 bits): register where result is stored.
Src1 (3 bits): second register input used by operation.
Imm / Src2 (6 bits): constant value or extra input field.

Register Guide
R0: accumulator (main result register).
R1: loop counter / frequently updated value.
R2: return/jump helper address.
R3-R7: general-purpose working registers.
