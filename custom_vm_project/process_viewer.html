About Process States

  • NEW: Process is created and enters the system.
  • READY: Waiting for CPU allocation.
  • RUNNING: Currently executing on CPU.
  • WAITING: Blocked, waiting for I/O or event.
  • TERMINATED: Finished execution and removed from system.

Each transition is triggered by real OS events (e.g., I/O, scheduling, completion).
